Description

If your motivation in the morning is low, you can create an app that sends you random phrases! In this stage, you will design a UI for this application. You have a list of phrases; create a button to add one more, and set up text with the reminder time.

Objectives

Create a screen that contains the following components:

  1. The TextView that shows whether the reminder was set, and if it was, at what time;

  2. The RecyclerView that contains your phrases

  3. A FloatingActionButton that allows you to add more phrases.

Set the following IDs for your widgets:

  • TextView: reminderTextView;

  • RecyclerView: recyclerView;

  • FloatingActionButton: addButton.

Set the reminderTextView text attribute to "No reminder set"
