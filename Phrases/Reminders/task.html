Description

The whole point of the app is to send you push notifications. Let's make it happen!

Objectives

  1. Create a notification channel for your app. Set the channel id to "org.hyperskill.phrases";

  2. On reminderTextView set an OnClickListener that shows a TimePickerDialog; we'll use it for setting the notification time. Don't reuse the same dialog for different clicks as this can cause undesired ux and problems with tests. Also make sure to use android.app.TimePickerDialog as this is the one tests are looking for;

  3. When a time is set update reminderTextView text to "Reminder set at $hour:$minute" where $hour and $minute are the hour and minute pick with TimePicker. Display the time using 24h format and always use two digits for $hour and $minute. (exs: "00:00", "09:45", "15:02", "23:59");

  4. Get an instance of AlarmManager, set an repeating alarm with the time set and with daily interval and send an explicit intent for a broadcast. If the time picked is before the current day time the alarm should be triggered only the next day. (ex: the current time is 15:00 and the time pick was 10:00);

  5. Create a class that extends BroadcastReceiver that will receive the broadcast from the alarm and send notifications when the broadcast is received. Add it to AndroidManifest;

  6. Create a notification with id 393939, the title Your phrase of the day and a random phrase from your ready-made phrases. That's why we asked you to have at least three phrases;
